# Community Event Webpage

This project is a structured HTML webpage for a fictional community event. The page demonstrates the use of semantic HTML elements to organize content clearly and accessibly.

## Features

- Header with event title and tagline
- Navigation menu linking to page sections
- Event details section
- Activities list
- Special guests displayed using a definition list
- Image section showcasing event photos
- Footer with contact information

## Concepts Used

- HTML5 semantic elements (header, nav, main, section, footer)
- Internal navigation using anchor links
- Unordered lists and definition lists
- Image embedding with descriptive alt text
- Basic webpage structure and accessibility practices

## Project Structure

index.html – main webpage structure  
apples.jpg – event image  
pumpkins.jpg – event image  

## How to View

1. Download or clone the repository
2. Open `index.html` in a web browser
